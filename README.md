# 介绍

很简单的插件，可以查看硬件和小软件信息，还可以运行系统命令
具体
#命令信息如下
- CPU型号
- CPU核心数
- 内存总大小
- 内存剩余大小
- 内存频率（不一定成功
- 系统版本
- Java进程数（可以大概知道这台服务器上运行了多少台mc服务器，如果是在docker环境下则获取不到
- 当前服务端启动参数
- 是否虚拟机（根据获取到的主板型号判断
- 当前权限（如 root
- 硬盘（大概率获取不到
- IP地址
 - IP具体位置
- 如果你想获取更多信息（我觉得这么多信息已经足够了），回帖告诉我


| 指令               | 权限 | 介绍                 |
| ------------------ | ---- | -------------------- |
| aida               | OP   | 显示服务器硬软件信息 |
| aida run <command> | OP   | 运行系统命令         |
