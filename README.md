# 介绍

很简单的一个小插件，可以查看服务器硬件和软件信息，还可以运行系统命令

# 命令

| 指令               | 权限 | 介绍                 |
| ------------------ | ---- | -------------------- |
| aida               | OP   | 显示服务器硬软件信息 |
| aida run <command> | OP   | 运行系统命令         |
